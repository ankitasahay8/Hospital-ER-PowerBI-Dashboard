# Hospital Emergency Room Dashboard (Power BI)

## Overview

This project is an interactive dashboard built using Power BI to analyze hospital emergency room data. The dashboard helps track patient visits, average waiting time, patient satisfaction, and department referrals.

## Tools Used

* Power BI
* Power Query
* DAX
* CSV Dataset

## Dashboard Features

* **Total Patients KPI**
* **Average Wait Time KPI**
* **Patient Satisfaction KPI**
* **Patients by Department**
* **Patients Over Time**
* **Patient Gender Distribution**
* **Interactive Department Filter**

## Insights

* Helps monitor patient flow in the emergency room.
* Identifies departments receiving the most patient referrals.
* Tracks trends in patient visits over time.
* Shows gender distribution of patients.

## Project Files

* `Hospital_ER_Dashboard.pbix` – Power BI dashboard
* `Hospital ER_Data.csv` – dataset used
* `dashboard.png` – dashboard preview

## Conclusion

This project demonstrates how Power BI can transform raw healthcare data into meaningful insights using interactive dashboards and visualizations.
